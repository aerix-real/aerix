# AERIX

Plataforma SaaS institucional de trading.

Stack:
- Node.js
- Express
- PostgreSQL
- JWT
- WebSocket
- React

Objetivo:
Criar plataforma hedge fund premium.

Regras:
- Nunca quebrar sistema existente
- Priorizar performance
- Código modular
- Responsividade obrigatória
- Design premium preto/dourado
